Crafting Guide for Minecraft +

v1.0.0

Built with web technologies using PhoneGap/cordova for Windows Phone and Android. Descriptions are from minecraft.gamepedia.com

This app shows you what materials you need and how to craft items using those materials in Minecraft. Crafting guides are included for basic items, tools, weapons, armour, food, mechanical, dyes, brewing, potions, banners and fireworks.

(This app is not official and is not affiliated with either Mojang or Microsoft).
